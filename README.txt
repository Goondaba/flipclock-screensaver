FlipClock

A simple flip clock screen saver written with SceneKit.
Requires at least 10.8 Mountain Lion.


Installing:

- Right-click the FlipClock.saver file, and select ‘Open’.
- At the security dialog, select ‘Open’
- This will open System Preferences and prompt for an install option.
- Enjoy!

Updating:

- Double-click the new version of the FlipClock.saver file.
- If you have System Preferences open while updating, restart System Preferences to see the changes take effect.
- Enjoy!
